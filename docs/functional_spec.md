# 宿泊税システム 機能仕様書（概要・ロジック）

## 1. システム概要
宿泊税の特別徴収義務者の管理、申告、賦課、徴収、および帳票発行を行うLGWAN-ASPシステム。

## 2. 画面構成・遷移
- **初期画面:** 特別徴収義務者管理台帳
  - 特別徴収義務者登録/編集
  - 宛名検索モーダル
  - 納税管理人登録/照会
  - 宿泊税納入金額管理台帳
    - 宿泊税情報登録/照会
  - 合算申告情報登録/照会
- **帳票発行:** 各種通知書の出力

## 3. 共通仕様
- **削除フラグ:** `del_flg = '0'` のデータのみを表示対象とする。
- **最新フラグ:** 履歴を持つテーブル（台帳等）は `new_flg = '1'` を最新データとして扱う。
- **日付管理:** 登録日は `add_dt`、更新日は `upd_dt` で管理し、システム日付を自動セットする。

## 4. 主要ロジック
- **宛名紐付け:** すべての義務者・施設情報は `m_atena`（宛名マスタ）の `atena_no` をキーとして関連付けられる。
- **検索処理:** - 施設名称、氏名などは部分一致検索。
  - 自治体コード（`jichitai_cd`）によるマルチテナント分離。
